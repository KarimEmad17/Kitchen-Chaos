One-line pitch: A fast-paced, time-pressured cooking simulation where you chop, fry, plate, and deliver dishes before the orders — and the clock — get the better of you.


Engine/Tools: Unity 2022 LTS, C#, State Machines, ScriptableObjects, UI/Audio Systems
Role: Solo Developer
Platform: PC
🎯 Challenge
Build a fast-paced, single-player cooking simulation with multiple interacting gameplay systems — recipes, cutting, frying, plating, delivery, and a countdown timer — while keeping the codebase clean and extensible.


🔧 Contribution
Implemented core gameplay using finite-state machines for player and game-flow states, an interaction system for picking up, placing, and processing kitchen objects, ScriptableObject-driven recipe and ingredient data, and dynamic UI for order tracking, scoring, and the countdown timer.

✅ Result
Delivered a complete, polished single-player gameplay loop with responsive controls, audio/UI feedback, and a recipe/scoring system architected to support new recipes and stations without code changes.
